Troubleshooting

If you have any trouble with updating your files, be sure to check the Requirements and the Prepare for updating sections.

Log File

If you received an error dialog box during the update, the error should be recorded in the log file (log.txt) located in the same directory as the application. When reporting issues, please refer to this log file.

Issue Reporting

If you'd like to report issues, please see this application's Github site.
